# Context Engineering Skill for Claude

> Master the art and science of context engineering - the critical discipline for building reliable, efficient, and powerful AI systems.

## 🎯 What is This?

This is a **custom skill for Claude** that teaches it advanced context engineering principles, patterns, and techniques. Context engineering is the practice of systematically designing and optimizing the information provided to AI systems - it's what separates hobbyist AI projects from production-grade systems.

## 🚀 Why Context Engineering?

**Prompt engineering tells AI *how* to respond. Context engineering determines *what* it knows.**

In 2026, context engineering has emerged as the defining skill for AI development:
- 🎯 **Better Results**: Properly engineered context leads to more accurate, relevant responses
- 💰 **Lower Costs**: Optimized context reduces token usage by 40-70%
- ⚡ **Faster Performance**: Smart caching and retrieval cut latency dramatically
- 🔧 **Reliable Systems**: Structured context makes AI behavior predictable and debuggable
- 📈 **Scalable Solutions**: Context engineering patterns work from prototype to production

## 📦 What's Included

- **Core Principles**: Context window optimization, information architecture, RAG, memory management
- **Design Patterns**: Layered architecture, progressive loading, compression, multi-modal integration
- **Anti-Patterns**: Common mistakes to avoid
- **Best Practices**: For both developers building AI systems and users working with AI assistants
- **Advanced Techniques**: Semantic chunking, context routing, adaptive windows, temporal weighting
- **Tools & Technologies**: Vector databases, embedding models, RAG frameworks, memory systems
- **Evaluation Metrics**: How to measure and improve context quality
- **Real-World Use Cases**: Software development, customer support, research, content creation, data analysis

## 📥 How to Install

### For Claude.ai (Web)
1. Download the `SKILL.md` file from this repository
2. Go to [claude.ai](https://claude.ai)
3. Click on your profile → **Skills**
4. Click **Create Skill** → **Upload**
5. Upload the `SKILL.md` file
6. The skill will now be available in all your conversations!

### For Claude Desktop / Kiro
1. Clone or download this repository
2. Copy `SKILL.md` to your skills directory:
   - **macOS/Linux**: `~/.claude/skills/` or `~/.kiro/skills/`
   - **Windows**: `%USERPROFILE%\.claude\skills\` or `%USERPROFILE%\.kiro\skills\`
3. Restart Claude Desktop / Kiro
4. The skill will be automatically loaded

### For Claude Code
1. Copy `SKILL.md` to your project's `.claude/skills/` directory
2. Or add it to your global skills folder
3. Reference it in your conversations with `#context-engineering`
