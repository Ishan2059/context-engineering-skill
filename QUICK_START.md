# Quick Start Guide

## 🚀 Get Started in 3 Minutes

### Step 1: Download the Skill
Download `SKILL.md` from this repository (click the file → Raw → Save)

### Step 2: Install in Claude

**For Claude.ai (Web):**
1. Go to [claude.ai](https://claude.ai)
2. Click your profile picture → **Skills**
3. Click **Create Skill** → **Upload**
4. Select the `SKILL.md` file
5. Done! ✅

**For Claude Desktop:**
1. Create folder: `~/.claude/skills/` (Mac/Linux) or `%USERPROFILE%\.claude\skills\` (Windows)
2. Copy `SKILL.md` into that folder
3. Restart Claude Desktop
4. Done! ✅

### Step 3: Try It Out

Start a new conversation and ask:

```
"Help me design a RAG system for my documentation"
```

or

```
"How should I structure context for an AI coding assistant?"
```

Claude will now respond with expert context engineering knowledge!

### For Developers

**Query:** "I'm building an AI agent that needs to remember user preferences across sessions. What's the best context architecture?"

**What you'll get:** Detailed memory management strategies, implementation patterns, code examples, and tool recommendations.

---

**Query:** "My RAG system retrieves 20 documents but only 2 are relevant. How do I improve precision?"

**Query:** "We want to add AI to our customer support. What context should we provide to the AI?"

### For Optimizing Costs

**Query:** "Our AI application costs $5000/month in API calls. Most of it is context tokens. How can we reduce this?"

## 🆘 Troubleshooting

**Q: The skill isn't working**  
A: Make sure the file is named exactly `SKILL.md` and is in the correct folder. Restart Claude if needed.

**Q: Claude doesn't seem to use the skill**  
A: The skill activates automatically when you ask context engineering questions. Try being more specific about context, RAG, or memory management.

**Q: Can I modify the skill?**  
A: Absolutely! It's MIT licensed. Edit `SKILL.md` to add your own techniques and patterns.

**Q: Does this work with other AI assistants?**  
A: The SKILL.md format is Claude-specific, but the principles and techniques apply to any AI system. You can adapt the content for ChatGPT, Gemini, etc.
